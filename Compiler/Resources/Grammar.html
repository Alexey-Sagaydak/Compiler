Грамматика

G[<ЦК> = <целочисленная константа>]:

VT = { ‘a’…’z’, ‘A’…’Z’, ‘0’…’9’, ‘:’, ‘;’, ‘+’, ‘-‘, ‘=’, ‘_’ }

VN = { <ЦК>, <константа>, <присваивание>, <число>, З, Б, Ц, E, ID }

P = {
 1. <ЦК> → ‘DECLARE’ E
 2. E → ID <константа> <число> ’;’
 3. ID → Б { Б | Ц | ‘_’ }
 4. <константа> → ‘CONSTANT’ ’INTEGER’ <присваивание>
 5. <число> → З [ Ц ]
 6. <присваивание> → ‘:=’ | ‘=’
 7. З → ‘+’ | ‘-‘ | ε
 8. Б → ‘a’ | ‘b’ | … | ‘z’ | ‘A’ | ‘B’ | … | ‘Z’
 9. Ц → ‘0’ | ‘1’ | … | ‘9’
}
